# 🛡️ AEGESIS
## AI-Powered Women Safety & Online Threat Intelligence Platform

> Predict. Protect. Respond. Prove.

AEGESIS is an AI-powered women safety platform designed to provide proactive protection against both physical and digital threats. Unlike traditional safety applications that only react after emergencies, AEGESIS leverages Artificial Intelligence to detect risks early, provide real-time safety assistance, automate emergency responses, and securely preserve evidence.

---

# 🌟 Problem Statement

Women today face interconnected safety challenges in both physical and digital environments.

Existing solutions mainly focus on emergency SOS features and lack:
- Proactive threat detection
- Online harassment and manipulation analysis
- Intelligent emergency response
- Secure evidence preservation
- Privacy-focused safety management

AEGESIS addresses these gaps by combining AI, real-time location services, and emergency management into one unified platform.

---

## 🔒 Prove

- Report Generation

## ❤️ Recovery

- Next-Step Recommendations

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/hriddhimabhatt9-jpg/Agesis-AI_powered_woem_safety_and_threat_intelligence_platform.git
```

Go to the project folder:

```bash
cd Agesis-AI_powered_woem_safety_and_threat_intelligence_platform
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
